# Estudos em Java

Este repositório reúne meus exercícios, anotações e mini-projetos desenvolvidos durante meus estudos em Java no curso de Análise e Desenvolvimento de Sistemas.

Conteúdo

- Lógica de programação
- Estruturas condicionais
- Laços de repetição
- Orientação a objetos (POO)
- Interfaces gráficas com Swing (em breve)


 Tecnologias usadas

- Java 17
- IDE: IntelliJ IDEA / VS Code
- Swing (para testes com interface gráfica)

Objetivo

Evoluir na linguagem Java, entendendo bem os conceitos de programação estruturada e orientada a objetos.

Organização do projeto

- logica-basica: exercícios de operadores e conceitos básicos
- estruturas: condicionais` → if, else, switch
- loops: for, while, do-while
- poo: classes, objetos, métodos
- interfaces-graficas` → projetos com Swing
